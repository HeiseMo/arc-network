## ğŸš¨ BOUNTY REPORTING RULES

### Mandatory Information:
1. **Session ID** (Format: XXXXX-XXXXX-XXXXX)
   - Must match active session where kill occurred
   - Case-sensitive
   - Must be verifiable

2. **Target Name** (Killer's Username)
   - Minimum 3 characters
   - Exact username required
   - No aliases or nicknames

3. **Last Known Location**
   - Specific area/landmark
   - Must be within game map
   - Example: "North Base", "Mining Sector 7"

### Optional but Recommended:
1. **Time Left in Session**
   - Minutes remaining when killed (0-300)
   - Helps hunters gauge urgency
   - Indicates if target still active

2. **Description**
   - Context of engagement
   - Target's tactics/behavior
   - Weapon used
   - Number of teammates

3. **Reward Offer**
   - In-game items, credits, services
   - Must honor agreements
   - Clearly specify conditions

4. **Visual Proof**
   - Screenshot or video of death
   - Kill feed visible
   - Max 10MB file size

## ğŸ¯ HUNTER ENGAGEMENT PROTOCOLS

### Phase 1: Target Acquisition
```
1. Browse active bounties in feed
2. Identify targets in your current/next session
3. Note time left in session (if available)
4. Check active period (24h/1week/unlimited)
```

### Phase 2: Session Entry
```
1. Join EXACT session from bounty listing
2. Verify target is present
3. Begin tracking
4. Maintain awareness of session time
```

### Phase 3: Engagement
```
1. Locate target using last known location
2. Engage only in fair combat
3. No teaming in solo modes
4. No exploits or cheats
```

### Phase 4: Confirmation
```
1. Take screenshot of kill feed
2. Note exact session ID
3. Click "CLAIM BOUNTY" on bounty modal
4. Upload proof immediately
5. Submit session ID
6. Wait for reporter approval
```
